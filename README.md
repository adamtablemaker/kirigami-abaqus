# Kirigami Buckling and Energy Dissipation (Abaqus)

This project studies the mechanical behavior of ribbon kirigami under axial tension using nonlinear finite element analysis.

## Problem

Thin sheets under tension remain in a trivial membrane state. Introducing kirigami cut patterns creates geometric degrees of freedom that enable controlled out-of-plane buckling and energy dissipation.

## Approach

* Eigenvalue buckling analysis to identify mode shapes
* Mode selection used as geometric imperfection
* Nonlinear Riks analysis to capture post-buckling response
* Mesh refinement and convergence verification

## Key Observations

* Mode selection strongly influences peak force and deformation path
* Buckling propagates sequentially through unit cells
* System exhibits a stable force plateau under large extension

## Result

The model captures geometry-driven instability and demonstrates how cut patterns control global force–displacement behavior.

## Ongoing Work

Extending from periodic ribbon systems to graded geometries and curved shells to program deformation pathways.
